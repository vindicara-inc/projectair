# Claims and Taxonomy Discipline (docs)

When editing docs or README files:

- Detector count: **"10 OWASP Agentic + 3 OWASP LLM + 3 AIR-native = 16 total."** Never "14" or "8 of 10."
- ASI10 is **declared-scope Zero-Trust enforcement**, NOT anomaly detection.
- AIR-04 (chain-integrity gap) is NOT ASI10 coverage. Do not conflate.
- Use "Project AIR" on hero pages, pitch decks, whitepapers, legal, press, investor materials.
- "Signed Intent Capsule" is the public-facing term for AgDR records.
- Every public claim must be grounded in an actual source document.
- Full detector detail lives in `docs/DETECTORS.md`. Read it before editing detector taxonomy.
